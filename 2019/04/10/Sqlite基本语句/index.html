Sqlite基本语句

后端 后端 数据库 2019/04/10

本文于 318 天之前发表，文中内容可能已经过时。

  • sqlite3 # 第一个命令 使用这个命令进入数据库
  • .open 数据库文件名.sqlite3 #打开新建的数据库文件
  • .read create.sql # 读取数据库命令
  • .table #显示数据库文件下的所有表单
  • drop table 数据库表单名; #删除表格命令
  • select * from 数据库表单名; 查询全部数据库表单
  • .header on # 这句命令是显示表头
  • .mode column #这句命令是选择模式 会让数据库的表单显示的好看一点
